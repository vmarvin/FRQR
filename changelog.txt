1.0.1.0 Добавлена экспериментальная реализация чтения данных из источников ODBC. Требования к файлу конфигурации и формату отдаваемых сервером данных описаны в документе odbc.txt.

1.0.0.0 Релиз. Реализована работа с несколькими энергоблоками.

0.8.0.0 Реализован отбор выборки с наименьшим количеством строк с плохим качеством.

0.7.0.0 Реализована оценка качества данных в выборках по нескольким источникам.

0.6.0.0 Реализованы работа с несколькими независимыми источниками, а также формирование признака качества для отдельных строк, как по признаку от источника данных, так и по превышению допустимого интервала изменения шумоподобных сигналов.

0.5.0.0 Проект переписан с нуля на языке C# (ветка nprch-ng). Теперь работает с архивными станциями ПТК Квинт-7 напрямую, без явного использования проектных данных. В архитектуру заложена возможность поддержки дополнительных типов источников данных.

0.1.0.0 Предварительная версия на C++ (ветка nprch). Работает только с OPC HDA сервером ПТК Квинт-7 и только с текущим выбранным в Квинтеграторе проектом. Нет обработки признаков качества.
